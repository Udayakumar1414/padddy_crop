# 🌾 AI Smart Paddy Farming Advisor

## 📌 Project Description

An AI-powered smart agriculture system for paddy farming using Raspberry Pi, sensors, Streamlit, Gemini AI, and Rice Science PDF knowledge retrieval.

The system helps farmers monitor field conditions and receive intelligent farming advisories based on:
- Sensor data
- Rice crop stages
- AI analysis
- Rice cultivation PDF knowledge

# 🚀 Features

- 🌡️ Real-time Temperature Monitoring
- 💧 Humidity Monitoring
- 🌱 Soil Moisture Analysis
- 🚰 Water Level Monitoring
- 📏 Plant Height Detection using Ultrasonic Sensor
- 🌾 Rice Growth Stage Prediction
- 🤖 AI-Based Farming Advisory
- 🦠 Disease Risk Detection
- 💦 Irrigation Recommendations
- 🌿 Fertilizer Suggestions
- 📚 Rice PDF Knowledge Retrieval
- ❓ Additional Farmer Query Support
- 🧠 RAG (Retrieval Augmented Generation)
- 👨‍🌾 Farmer-Friendly Interface

# ⚙️ Working Principle

1. 📡 Sensors collect field data
2. 🍓 Raspberry Pi processes sensor values
3. 📊 Streamlit dashboard displays data
4. 🤖 Gemini AI analyzes crop conditions
5. 📚 Rice PDF knowledge is retrieved
6. 🧠 AI generates intelligent farming advisory
7. 👨‍🌾 Farmer receives recommendations

# 🤖 AI Integration

The project uses Gemini AI along with Rice Science PDF documents to provide intelligent farming recommendations based on:

- 🌡️ Current sensor values
- 🌾 Crop growth stage
- 📚 Rice cultivation knowledge
- ❓ Farmer queries

# 🧠 RAG Implementation

The project uses Retrieval Augmented Generation (RAG):

- 📄 Rice PDF documents are stored in ChromaDB
- 🔍 Relevant information is retrieved based on farmer queries
- 🤖 AI generates accurate responses using retrieved knowledge
